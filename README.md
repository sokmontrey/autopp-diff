## Getting Start
### Installation with CMake:

1. Clone the project:

```bash
git clone https://github.com/sokmontrey/nodeflow.git
```

2. Link the library:

```cmake
add_subdirectory(nodeflow)

target_link_libraries(${PROJECT_NAME} nodeflow)
```
